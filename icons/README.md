# Icons

Placeholder icons generated by `scripts/make-icons.py` (pure Python stdlib, no
dependencies).

- `icon-16.png`, `icon-32.png`, `icon-48.png`, `icon-128.png`

Regenerate with:

```bash
python3 scripts/make-icons.py
```

For a store submission, replace these with designed assets at the same sizes and
filenames. Chrome/Edge require PNG for `action.default_icon`; Firefox and Safari
also accept PNG, so PNG at all four sizes is the safe, portable choice.
